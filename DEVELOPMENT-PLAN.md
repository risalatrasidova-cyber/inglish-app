## 📋 ЭТАП 1: Настройка проекта (День 1)

### 1.1 Выбор технологического стека

**Рекомендации:**

**Backend:**
- **NestJS** (TypeScript, структурированный, с валидацией из коробки)
- **TypeORM** (для работы с БД)
- **PostgreSQL** или **MySQL** (для продакшена)
- **JWT** (для аутентификации)
- **class-validator** (для валидации)
- **Swagger** (для документации API)

**Frontend:**
- **React** + **TypeScript**
- **Vite** (для сборки)
- **React Query** (для работы с API)
- **React Router** (для роутинга)
- **Tailwind CSS** или **Material-UI** (для UI)

**Альтернативы:**
- Backend: Express + TypeScript
- Frontend: Vue 3 + TypeScript

### 1.2 Создать структуру проекта

```
inglish_app/
├── backend/          # NestJS приложение
│   ├── src/
│   │   ├── modules/
│   │   │   ├── auth/
│   │   │   ├── lessons/
│   │   │   ├── words/
│   │   │   ├── users/
│   │   │   ├── admin/
│   │   │   └── progress/
│   │   ├── common/
│   │   └── config/
│   ├── migrations/
│   └── package.json
├── frontend/         # React приложение
│   ├── src/
│   │   ├── components/
│   │   ├── pages/
│   │   ├── services/
│   │   ├── hooks/
│   │   └── types/
│   └── package.json
├── database-schema.md
├── api-endpoints.md
└── README.md
```

### 1.3 Инициализация проектов

**Backend:**
```bash
cd backend
npm install -g @nestjs/cli
nest new . --skip-git
npm install @nestjs/typeorm typeorm pg
npm install @nestjs/jwt @nestjs/passport passport passport-jwt
npm install class-validator class-transformer
npm install @nestjs/swagger swagger-ui-express
npm install bcrypt
npm install @nestjs/throttler
```

**Frontend:**
```bash
cd frontend
npm create vite@latest . -- --template react-ts
npm install
npm install react-router-dom
npm install @tanstack/react-query
npm install axios
npm install tailwindcss
```
