# Описание приложения для изучения английского языка

## 🎯 Общая концепция
Игровое приложение_ тренажер для изучения английского языка, предназначенное преимущественно для детей. Приложение имеет игровую тематику и систему вознаграждений.

---

## 🔐 Авторизация
- **Вход только по логину** (без пароля)
-
- **Прогресс НЕ сбрасывается между сессиями** - все достижения, деньги, звезды сохраняются

---

## 📱 Главный экран

### Элементы интерфейса:
- **Список уроков** - основной контент экрана
  - Пройденные уроки отображаются **зеленым цветом**
  - Под каждым уроком отображаются **звезды** (золотые/бриллиантовые) за достижения 90% и 100%
- **Иконка денег (вознаграждение)** - в правом верхнем углу, показывает накопленные деньги (общие для всех уровней и уроков)
- **Иконка пользователя** - в правом верхнем углу (иконка) и внизу логин пользователя
  - При нажатии открывается меню:
    - **Сменить пользователя**
    - **Добавить пользователя**
    - **Выход** 

### Логика открытия уроков:
- **Первый урок** - всегда активен (доступен для прохождения)
- **Последующие уроки** - открываются после прохождения **двух уровней** в предыдущем уроке

---

## 📚 Структура урока

Каждый урок содержит **3 уровня** (оформлены в игровой тематике):

1. **Уровень 1: Тест** - выбор правильного ответа из вариантов ⬅️ **ДЕТАЛЬНО ОПИСАНО НИЖЕ**
2. **Уровень 2: Аудио** - работа с аудио (описание позже)
3. **Уровень 3: Написание слов** - написание слов (описание позже)

### 🔄 Логика перехода между уровнями внутри урока:

**Цели и переходы:**
- **70% правильных ответов** → открывает переход на следующий уровень
- **90% правильных ответов** → золотая звезда за уровень + бонус 100₽ (однократно)
- **100% правильных ответов** → бриллиантовая звезда + бонус 150₽ (однократно)

**Условие для перехода к следующему уровню:**
- Пользователь должен набрать **минимум 70% правильных ответов** от общего количества слов в уровне
- **НЕ обязательно проходить все слова** - переход доступен при достижении 70%, даже если остались непройденные слова
- После достижения 70% показывается модальное окно с предложением перейти к следующему уровню
- Слова, которые не были пройдены, остаются доступными для прохождения (чтобы получить вознаграждение и звезды)
- Пользователь может вернуться к уровню позже и пройти оставшиеся слова
- **Прогресс накапливается между сессиями, НЕ сбрасывается!**

**Последовательность:**
- Уровень 1 → Уровень 2: при достижении 70%+ правильных ответов (не обязательно проходить все слова)
- Уровень 2 → Уровень 3: при достижении 70%+ правильных ответов (не обязательно проходить все слова)
- Уровень 3 → Следующий урок: при достижении 70%+ правильных ответов (не обязательно проходить все слова)

**Логика открытия следующих уроков:**
- Следующий урок открывается после прохождения **двух уровней** в предыдущем уроке
- Например: если пройдены Уровень 1 и Уровень 2 в Уроке 1, то Урок 2 становится доступным

---

## 🎮 УРОВЕНЬ 1: ТЕСТ (выбор правильного ответа)

### 📊 Общая информация:
- **Количество слов в уроке**: разное (зависит от урока)
- **Ограничение по времени**: нет
- **Вознаграждение за правильный ответ**: **5 рублей** (только за первое правильное прохождение слова в каждом уровне)
- **Прогресс НЕ сбрасывается между сессиями** - все достижения сохраняются
- **Текущая сессия** - при перезаходе в уровень начинается новая случайная последовательность слов 

---

### 🖥️ Интерфейс уровня:

#### **Верхняя часть страницы (меню):**

1. **Кнопка "Сбросить прогресс"**
   - Сбрасывает прогресс **текущей сессии уровня** (не всего урока)
   - Сбрасывает счетчики правильных/неправильных ответов для текущей сессии
   - Сбрасывает прогресс-бар уровня (но не общий прогресс по словам)
   - **НЕ сбрасывает:**
     - Накопленные деньги (вознаграждения)
     - Звезды за достижения
     - Историю правильных ответов по словам (какие слова уже пройдены правильно)
     - Общий прогресс между сессиями

2. **Кнопка "Все слова"**
   - Открывает список всех слов текущего урока
   - Показывается перевод на русский язык
   - **Цветовая индикация:**
     - Изначально все слова **серого цвета**
     - Правильно отвеченные слова становятся **зелеными**
     - Неправильно отвеченные остаются **серыми**
   - **Функционал:** При нажатии на слово воспроизводится аудио на английском языке
   - Нельзя перейти к конкретному слову для прохождения (только просмотр и прослушивание)

3. **Иконка вознаграждения (деньги)**
   - Показывает накопленные деньги (общие для всех уровней и уроков)
   - При клике открывает модальное окно: "Ты заработал [количество] денег"

4. **Кружок правильных ответов**
   - Показывает количество правильных ответов **только для текущего уровня**
   - Сбрасывается при нажатии "Сбросить прогресс"

5. **Кружок неправильных ответов**
   - Показывает количество неправильных ответов **только для текущего уровня**
   - Сбрасывается при нажатии "Сбросить прогресс"

6. **Прогресс-бар уровня** ⭐ **НОВОЕ**
   - Расположен в верхней части страницы, **всегда виден**
   - Показывает процент правильных ответов **от общего количества слов в уровне** (не от пройденных)
   - Формат: `[████████░░] 70% (7/10 правильных)`
   - Пример: если в уровне 10 слов, и пользователь правильно ответил на 7 - это 70%
   - **Динамически обновляется** после каждого ответа
   - Зеленая часть - правильные ответы
   - Серая часть - показывает, сколько еще нужно для 70%
   - При достижении **70%** прогресс-бар показывает зеленый цвет и **автоматически появляется модальное окно** с предложением перейти дальше
   - Показывает, достаточно ли правильных ответов для перехода к следующему уровню
   - **Важно:** Переход доступен при 70%, даже если не все слова пройдены 
#### **Центральная часть (основной контент):**

- **Вопрос (слово)**
  - Показывается слово согласно **алгоритму приоритетов** (см. раздел "Алгоритм показа слов")
  - Может быть показано **на английском** или **на русском** языке (случайно)
  - Если слово показано на английском → варианты ответов на русском
  - Если слово показано на русском → варианты ответов на английском
  - Оформления нет (только текст)
  - **Кнопка динамика** 🔊 рядом со словом - позволяет прослушать слово перед выбором ответа (аудио на английском языке)
  - Аудио на данном уровне воспроизводится только на английском языке

- **6 вариантов ответа**
  - Расположены в **2 вертикальных ряда**, по **3 слова в каждом ряду**
  - Один правильный ответ + 5 случайных из других слов того же урока
  - Если вопрос на английском → варианты на русском
  - Если вопрос на русском → варианты на английском 

### ⚙️ Механика проверки:

#### **Процесс ответа:**
1. Пользователь видит слово (вопрос)
2. Пользователь видит 6 вариантов ответа
3. Пользователь выбирает один из вариантов
4. Проверка происходит **автоматически** при выборе ответа

#### **Правильный ответ на НОВОЕ слово** (слово еще НИКОГДА не было пройдено правильно):

- Выбранный вариант подсвечивается **зеленым цветом**
- Воспроизводится **аудио на английском языке** (произношение слова)
- Показывается сообщение: **"+5₽"** (только первый раз для этого слова в этом уровне!)
- Начисляется **5 рублей** (однократно за это слово в этом уровне)
- Слово в списке "Все слова" становится **зеленым**
- Прогресс-бар **растет** (увеличивается процент правильных ответов)
- Автопереход через **2 секунды** к следующему слову
- Кнопки "Далее" нет

#### **Правильный ответ на УЖЕ ПРОЙДЕННОЕ слово** (слово уже было пройдено правильно ранее):

- Выбранный вариант подсвечивается **зеленым цветом**
- Показывается сообщение: **"Верно!"** (без денег)
- **Деньги НЕ начисляются** (уже получены ранее)
- Прогресс-бар **растет** (увеличивается процент правильных ответов)
- Автопереход через **2 секунды** к следующему слову

#### **Неправильный ответ:**

- Выбранный вариант подсвечивается **красным цветом**
- Показывается **правильный ответ** (подсвечивается зеленым)
- Показывается сообщение: **"Попробуй еще раз"**

- Слово в списке "Все слова" остается **серым** (или становится серым, если было зеленым)
- Прогресс-бар **НЕ меняется** (процент правильных ответов не увеличивается)
- Автопереход через **2 секунды** к следующему слову
- Кнопки "Попробовать еще раз" нет

### 🏁 Завершение уровня и переход:

#### **При выходе из уровня:**

**Модальное окно при выходе:**
- Текст: "Сегодня ты выучил X новых слов!"
- Текст: "Всего в уровне: Y/Z слов"
- Текст: "Заработано сегодня: N₽"
- Кнопки:
  - **"Продолжить"** (остаться в уровне)
  - **"Выйти"** (вернуться к списку уроков)

#### **При достижении 70% правильных ответов:**

**Модальное окно появляется автоматически:**
- Текст: "Уровень пройден! (70%)"
- Текст: "Открыт переход к следующему уровню!"
- Текст: "Осталось слов до звёзд: K" (сколько слов нужно для 90% и 100%)
- Кнопки:
  - **"Следующий уровень"** (активна, зеленого цвета)
  - **"Остаться тут"** (продолжить прохождение для получения звезд)

#### **При достижении 90% правильных ответов:**

**Модальное окно появляется автоматически:**
- Текст: "Поздравляем! 🎉"
- Текст: "Золотая звезда +100₽!"
- Текст: "Ты знаешь 90% слов этого уровня!"
- Начисляется **бонус 100₽** (однократно)
- Золотая звезда отображается на главном экране под уроком
- Кнопки:
  - **"Ура!"** (закрыть модалку)
  - **"Посмотреть звёзды"** (перейти к просмотру достижений)

#### **При достижении 100% правильных ответов:**

**Модальное окно появляется автоматически:**
- Текст: "Идеально! 💎"
- Текст: "Бриллиантовая звезда +150₽!"
- Текст: "Ты прошёл уровень на 100%!"
- Начисляется **бонус 150₽** (однократно)
- Бриллиантовая звезда отображается на главном экране под уроком
- Кнопки:
  - **"Супер!"** (закрыть модалку)
  - **"Главное меню"** (вернуться к списку уроков)

**Важные детали:**
- Модалки появляются автоматически при достижении соответствующих процентов
- Бонусы за 90% и 100% начисляются **однократно** (даже если пользователь сбросит прогресс)
- Звезды отображаются на главном экране под уроком
- Прогресс накапливается между сессиями, НЕ сбрасывается

### 💰 Система вознаграждений (деньги):

**Деньги начисляются:**
- **+5₽** — за первое правильное прохождение слова в каждом уровне (однократно)
- **+100₽** — бонус за достижение 90% в уровне (однократно)
- **+150₽** — бонус за достижение 100% в уровне (однократно)

**Важные правила:**
- Деньги **никогда не сбрасываются** — общий счетчик для всех уровней и уроков
- За одно слово можно получить **5₽ только один раз** в каждом уровне (даже если сбросить прогресс)
- Бонусы за 90% и 100% начисляются **однократно** за уровень (даже если сбросить прогресс)
- Деньги - **просто счетчик достижений** (нельзя тратить)
- Отслеживается через отдельную таблицу в БД с отметками о полученных вознаграждениях

### ⭐ Система звезд (визуальные достижения):

**Звезды начисляются:**
- **Золотая звезда** — за достижение 90% правильных ответов в уровне (однократно)
- **Бриллиантовая звезда** — за достижение 100% правильных ответов в уровне (однократно)

**Отображение:**
- Звезды отображаются на **главном экране под уроком**
- Золотая звезда заменяется на бриллиантовую при достижении 100%
- Звезды **никогда не сбрасываются** — сохраняются между сессиями

- Слова хранятся в базе данных

- Каждое слово имеет:
  - Английское написание
  - Русский перевод
  - Аудио произношение (на английском) - формат **MP3**

- Английское слово (текст)
- Русский перевод (текст)
- Аудио файл (произношение) - формат MP3
- Принадлежность к уроку
- Категории/теги не нужны - категоризация происходит через название урока

### 📊 Прогресс и статистика:

#### **Визуальная индикация прогресса:**
- Пройденные уровни отображаются **зеленым цветом**
- Пройденные уроки отображаются **зеленым цветом**

#### **Сохранение прогресса в БД:**

**Прогресс сохраняется автоматически и НЕ сбрасывается между сессиями.**

##### **A. Основной прогресс по уровню (обязательно сохраняем НАВСЕГДА):**

Для каждого пользователя и каждого уровня сохраняется:

1. **Лучший достигнутый процент** (например: 70%, 90%, 100%)
   - Максимальный процент правильных ответов, который когда-либо был достигнут

2. **Сколько всего правильных ответов дал за ВСЁ время в этом уровне**
   - Общий счетчик всех правильных ответов за все сессии

3. **Сколько всего попыток было за ВСЁ время**
   - Общий счетчик всех попыток (правильных + неправильных) за все сессии

4. **Статус "пройден"** (достигнут ли 70%)
   - Булево значение: true/false
   - Определяет, открыт ли переход к следующему уровню

5. **Получена ли уже золотая звезда** (за 90%)
   - Булево значение: true/false
   - Однократно, не сбрасывается

6. **Получен ли уже бриллиант** (за 100%)
   - Булево значение: true/false
   - Однократно, не сбрасывается

7. **Когда впервые начал уровень**
   - Дата и время первого входа в уровень

8. **Когда последний раз играл**
   - Дата и время последнего взаимодействия с уровнем

##### **B. По каждому слову в каждом уровне отдельно (обязательно сохраняем НАВСЕГДА):**

Для каждого пользователя, каждого слова и каждого уровня сохраняется:

1. **Факт: "Это слово хоть раз было пройдено правильно в этом уровне"**
   - Булево значение: true/false
   - Используется для алгоритма приоритетов (Приоритет 1)

2. **Сколько раз правильно отвечал на это слово**
   - Счетчик правильных ответов по конкретному слову

3. **Когда впервые правильно ответил на это слово**
   - Дата и время первого правильного ответа

4. **Когда последний раз правильно ответил**
   - Дата и время последнего правильного ответа

##### **C. Вознаграждения (самое важное! Обязательно сохраняем НАВСЕГДА):**

**Отдельная таблица**, где каждая запись = факт получения денег.

**Структура записи:**
- Пользователь ID
- Тип вознаграждения (слово / 90% / 100%)
- Уровень ID (если применимо)
- Слово ID (если применимо)
- Сумма (5₽ / 100₽ / 150₽)
- Дата и время получения
- **Уникальный ключ** (комбинация полей), чтобы нельзя было получить дважды за одно и то же

**Примеры записей:**
- "Пользователь X получил 5₽ за слово Y в уровне Z" (однократно)
- "Пользователь X получил 100₽ за достижение 90% в уровне Z" (однократно)
- "Пользователь X получил 150₽ за достижение 100% в уровне Z" (однократно)

##### **НЕ сохраняем (временные данные):**

1. **Текущую позицию в уровне** (какое слово показывать следующим)
   - При перезаходе начинается новая случайная последовательность

2. **Порядок слов в текущей сессии**
   - Каждая сессия генерируется заново по алгоритму приоритетов

3. **Временные счётчики за текущий заход**
   - Счетчики правильных/неправильных ответов для текущей сессии
   - Сбрасываются при выходе из уровня

**Важно:**
- Все данные сохраняются **НАВСЕГДА** и не сбрасываются
- При перезаходе в уровень начинается **новая случайная последовательность** слов (текущая сессия)
- Прогресс накапливается между сессиями 
---

### 📝 Дополнительные детали первого уровня:

1. **Алгоритм показа слов (приоритеты):**

   **Приоритет 1:** Слова, которые ещё **НИКОГДА не были пройдены правильно**
   - Сначала показываются новые слова, которые пользователь еще не знает
   
   **Приоритет 2:** Слова с **неправильными ответами в прошлом**
   - Затем показываются слова, на которые пользователь ранее отвечал неправильно
   
   **Приоритет 3:** **Случайные для повторения**
   - В конце показываются уже пройденные слова для закрепления (случайный порядок)

   **Важно:**
   - Не обязательно проходить все слова для перехода к следующему уровню (достаточно 70% правильных ответов)
   - Слова, которые не были пройдены, остаются доступными для прохождения (для получения вознаграждения)
   - При перезаходе в уровень начинается **новая случайная последовательность** слов (текущая сессия)

2. **Повторное прохождение:**
   - Если пользователь сбросил прогресс, все слова снова становятся доступными
   - Можно проходить слова повторно без сброса прогресса (после завершения уровня можно выбрать "Остаться")

3. **Варианты ответов:**
   - 5 неправильных вариантов выбираются **случайно из других слов того же урока**
   - Правильный ответ не может повторяться в вариантах

4. **Аудио:**
   - Аудио воспроизводится **только после правильного ответа** (на английском языке)
   - До выбора ответа аудио не воспроизводится автоматически
   - Можно прослушать аудио в списке "Все слова" (при нажатии на слово)
   - Можно прослушать аудио через кнопку динамика рядом со словом (перед выбором ответа)

5. **Счетчики правильных/неправильных ответов:**
   - Сбрасываются при сбросе прогресса
   - Показывают только текущую сессию уровня

6. **Список "Все слова":**
   - Только просмотр статуса и прослушивание аудио
   - Нельзя начать прохождение конкретного слова из списка

## 📝 Примечания для разработки

- Приложение должно быть **игровым и привлекательным для детей**
- Все элементы должны быть **интуитивно понятными**
- **Визуальная обратная связь** важна (цвета, сообщения)
- Анимации не нужны
- Персонажи не нужны
- Звуковые эффекты пока не нужны (только аудио произношения слов)
- Система вознаграждений должна **мотивировать** к продолжению обучения
