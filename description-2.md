# УРОВЕНЬ 2: АУДИО (произношение слов)

## 📊 Общая информация:
- **Количество слов в уроке**: разное (зависит от урока) - **те же слова, что и в Уровне 1**
- **Ограничение по времени**: нет
- **Вознаграждение за правильный ответ**: **5 рублей** (только за первое правильное прохождение слова в каждом уровне)
- **Прогресс НЕ сбрасывается между сессиями** - все достижения сохраняются
- **Текущая сессия** - при перезаходе в уровень начинается новая случайная последовательность слов
- **Логика идентична Уровню 1**, кроме основного контента (вместо выбора ответа - произношение)

---

## 🖥️ Интерфейс уровня:

### **Верхняя часть страницы (меню):**

**Все элементы меню идентичны Уровню 1:**

1. **Кнопка "Сбросить прогресс"**
   - Сбрасывает прогресс **текущей сессии уровня** (не всего урока)
   - Сбрасывает счетчики правильных/неправильных ответов для текущей сессии
   - Сбрасывает прогресс-бар уровня (но не общий прогресс по словам)
   - **НЕ сбрасывает:**
     - Накопленные деньги (вознаграждения)
     - Звезды за достижения
     - Историю правильных ответов по словам (какие слова уже пройдены правильно)
     - Общий прогресс между сессиями

2. **Кнопка "Все слова"**
   - Открывает список всех слов текущего урока
   - Показывается перевод на русский язык
   - **Цветовая индикация:**
     - Изначально все слова **серого цвета**
     - Правильно отвеченные слова становятся **зелеными**
     - Неправильно отвеченные остаются **серыми**
   - **Функционал:** При нажатии на слово воспроизводится аудио на английском языке
   - Нельзя перейти к конкретному слову для прохождения (только просмотр и прослушивание)

3. **Иконка вознаграждения (деньги)**
   - Показывает накопленные деньги (общие для всех уровней и уроков)
   - При клике открывает модальное окно: "Ты заработал [количество] денег"

4. **Кружок правильных ответов**
   - Показывает количество правильных ответов **только для текущего уровня**
   - Сбрасывается при нажатии "Сбросить прогресс"

5. **Кружок неправильных ответов**
   - Показывает количество неправильных ответов **только для текущего уровня**
   - Сбрасывается при нажатии "Сбросить прогресс"

6. **Прогресс-бар уровня**
   - Расположен в верхней части страницы, **всегда виден**
   - Показывает процент правильных ответов **от общего количества слов в уровне** (не от пройденных)
   - Формат: `[████████░░] 70% (7/10 правильных)`
   - Пример: если в уровне 10 слов, и пользователь правильно ответил на 7 - это 70%
   - **Динамически обновляется** после каждого ответа
   - Зеленая часть - правильные ответы
   - Серая часть - показывает, сколько еще нужно для 70%
   - При достижении **70%** прогресс-бар показывает зеленый цвет и **автоматически появляется модальное окно** с предложением перейти дальше
   - Показывает, достаточно ли правильных ответов для перехода к следующему уровню
   - **Важно:** Переход доступен при 70%, даже если не все слова пройдены

### **Центральная часть (основной контент):**

#### **Отображение слова:**
- Показывается слово согласно **алгоритму приоритетов** (см. раздел "Алгоритм показа слов")
- Слово показывается **на русском языке** (перевод)
- Пользователь должен произнести слово **на английском языке**
- Оформления нет (только текст)
- **Кнопка динамика** 🔊 рядом со словом - позволяет прослушать правильное произношение перед ответом (аудио на английском языке)

#### **Иконка микрофона:**
- Расположена **по центру** (там, где в Уровне 1 были варианты ответов)
- Большая, заметная иконка микрофона 🎤
- При нажатии на иконку:
  - Иконка меняет цвет/состояние (визуальная индикация записи)
  - Начинается запись аудио с микрофона пользователя
  - Пользователь произносит слово на английском языке
  - При повторном нажатии (или автоматически после паузы) запись останавливается
  - Происходит автоматическая проверка произношения

#### **Визуальная обратная связь во время записи:**
- Иконка микрофона подсвечивается/пульсирует (индикация активной записи)
- Можно показать анимацию звуковых волн (опционально)
- Таймер записи (опционально, но не обязательно)

## ⚙️ Механика проверки произношения:

### **Процесс ответа:**
1. Пользователь видит слово на русском языке
2. Пользователь может прослушать правильное произношение (кнопка динамика)
3. Пользователь нажимает на иконку микрофона
4. Начинается запись аудио
5. Пользователь произносит слово на английском языке
6. Запись останавливается (автоматически или при повторном нажатии)
7. Происходит **автоматическая проверка** произношения через Speech Recognition API
8. Результат проверки отображается пользователю

### **Правильный ответ на НОВОЕ слово** (слово еще НИКОГДА не было пройдено правильно):

- Показывается сообщение: **"+5₽"** (только первый раз для этого слова в этом уровне!)
- Начисляется **5 рублей** (однократно за это слово в этом уровне)
- Воспроизводится **аудио на английском языке** (правильное произношение слова)
- Слово в списке "Все слова" становится **зеленым**
- Прогресс-бар **растет** (увеличивается процент правильных ответов)
- Автопереход через **2 секунды** к следующему слову
- Кнопки "Далее" нет

### **Правильный ответ на УЖЕ ПРОЙДЕННОЕ слово** (слово уже было пройдено правильно ранее):

- Показывается сообщение: **"Верно!"** (без денег)
- **Деньги НЕ начисляются** (уже получены ранее)
- Воспроизводится **аудио на английском языке** (правильное произношение слова)
- Прогресс-бар **растет** (увеличивается процент правильных ответов)
- Автопереход через **2 секунды** к следующему слову

### **Неправильный ответ:**

- Показывается сообщение: **"Попробуй еще раз"**
- Воспроизводится **аудио на английском языке** (правильное произношение для обучения)
- Показывается правильное написание слова на английском языке (опционально)
- Слово в списке "Все слова" остается **серым** (или становится серым, если было зеленым)
- Прогресс-бар **НЕ меняется** (процент правильных ответов не увеличивается)
- Автопереход через **2 секунды** к следующему слову
- Кнопки "Попробовать еще раз" нет (можно повторить попытку при следующем показе этого слова)

## 🎤 Варианты реализации проверки аудио (Speech Recognition):

### **Вариант 1: Web Speech API (браузерный, бесплатный)** ⭐ **ВЫБРАН ДЛЯ РЕАЛИЗАЦИИ**

**Описание:**
- Встроенный API браузера (Chrome, Edge, Safari)
- Не требует серверной обработки
- Бесплатный
- Работает полностью на клиенте

**Плюсы:**
- ✅ Бесплатно
- ✅ Не требует серверной обработки
- ✅ Быстрая проверка (без задержек сети)
- ✅ Простая интеграция
- ✅ Не требует API ключей

**Минусы:**
- ❌ Может быть менее точным для акцентов
- ❌ Зависит от качества микрофона
- ❌ Может требовать разрешения браузера на доступ к микрофону
- ❌ Разная точность в разных браузерах

**Реализация:**
```javascript
// Пример использования Web Speech API
const recognition = new webkitSpeechRecognition() || new SpeechRecognition();
recognition.lang = 'en-US';
recognition.continuous = false;
recognition.interimResults = false;

recognition.onresult = (event) => {
  const spokenWord = event.results[0][0].transcript.toLowerCase().trim();
  const correctWord = expectedWord.toLowerCase().trim();
  
  // Проверка с учетом порога 80%
  const similarity = calculateSimilarity(spokenWord, correctWord); // Используя Levenshtein distance или аналогичный алгоритм
  const threshold = 0.80; // 80% порог совпадения
  
  return similarity >= threshold;
};
```

**Точность проверки:**
- Используется **fuzzy matching** (нечеткое сравнение) для учета акцентов
- Сравнивать не только точное совпадение, но и фонетическое сходство
- Учитывать возможные опечатки в распознавании
- **Порог совпадения: 80%** - если распознанный текст совпадает на 80%+ с правильным ответом → засчитывается

- ❌ Более сложная реализация

6. Результат возвращается клиенту

## 🔍 Алгоритм проверки произношения:

### **Точное совпадение:**
- Распознанный текст точно совпадает с правильным ответом (без учета регистра)
- Пример: "hello" = "hello" ✅

### **Fuzzy matching (нечеткое сравнение):**
- Учитывает возможные опечатки в распознавании
- Учитывает акценты и вариации произношения
- Используется библиотека для сравнения строк (например, Levenshtein distance)

**Примеры допустимых совпадений:**
- "hello" = "hello" ✅ (точное)
- "helo" = "hello" ✅ (опечатка, но близко)
- "hellow" = "hello" ✅ (опечатка, но близко)
- "cat" ≠ "dog" ❌ (разные слова)

**Порог совпадения:**
- **Установлен порог: 80% совпадения** ✅
- Если распознанный текст совпадает на **80% или более** с правильным ответом → засчитывается как правильный ответ
- Если совпадение менее 80% → засчитывается как неправильный ответ

**Примеры проверки с порогом 80%:**
- "hello" vs "hello" = 100% → ✅ Правильно
- "helo" vs "hello" = ~83% → ✅ Правильно (превышает 80%)
- "hellow" vs "hello" = ~83% → ✅ Правильно (превышает 80%)
- "cat" vs "dog" = ~0% → ❌ Неправильно (ниже 80%)
- "cat" vs "cat" = 100% → ✅ Правильно

### **Фонетическое сравнение (опционально, для продвинутой версии):**
- Сравнение не только текста, но и звучания
- Учитывает акценты и произношение
- Требует более сложную реализацию

**Важные детали:**

- Звезды отображаются на главном экране под уроком
- Прогресс накапливается между сессиями, НЕ сбрасывается

## 💰 Система вознаграждений (деньги):

**Деньги начисляются:**
- **+5₽** — за первое правильное прохождение слова в каждом уровне (однократно)
- **+100₽** — бонус за достижение 90% в уровне (однократно)
- **+150₽** — бонус за достижение 100% в уровне (однократно)

**Важные правила:**
- Деньги **никогда не сбрасываются** — общий счетчик для всех уровней и уроков
- За одно слово можно получить **5₽ только один раз** в каждом уровне (даже если сбросить прогресс)
- Бонусы за 90% и 100% начисляются **однократно** за уровень (даже если сбросить прогресс)
- Деньги - **просто счетчик достижений** (нельзя тратить)
- Отслеживается через отдельную таблицу в БД с отметками о полученных вознаграждениях

### ⭐ Система звезд (визуальные достижения):

**Звезды начисляются:**
- **Золотая звезда** — за достижение 90% правильных ответов в уровне (однократно)
- **Бриллиантовая звезда** — за достижение 100% правильных ответов в уровне (однократно)

**Отображение:**
- Звезды отображаются на **главном экране под уроком**
- Золотая звезда заменяется на бриллиантовую при достижении 100%
- Звезды **никогда не сбрасываются** — сохраняются между сессиями

## 📝 Алгоритм показа слов:

**Приоритеты идентичны Уровню 1:**

**Приоритет 1:** Слова, которые ещё **НИКОГДА не были пройдены правильно**
- Сначала показываются новые слова, которые пользователь еще не знает

**Приоритет 2:** Слова с **неправильными ответами в прошлом**
- Затем показываются слова, на которые пользователь ранее отвечал неправильно

**Приоритет 3:** **Случайные для повторения**
- В конце показываются уже пройденные слова для закрепления (случайный порядок)

**Важно:**
- Не обязательно проходить все слова для перехода к следующему уровню (достаточно 70% правильных ответов)
- Слова, которые не были пройдены, остаются доступными для прохождения (для получения вознаграждения)
- При перезаходе в уровень начинается **новая случайная последовательность** слов (текущая сессия)

## 📊 Прогресс и статистика:

### **Визуальная индикация прогресса:**
- Пройденные уровни отображаются **зеленым цветом**
- Пройденные уроки отображаются **зеленым цветом**

### **Сохранение прогресса в БД:**

**Прогресс сохраняется автоматически и НЕ сбрасывается между сессиями.**

**Структура сохранения идентична Уровню 1:**

##### **A. Основной прогресс по уровню (обязательно сохраняем НАВСЕГДА):**
- Лучший достигнутый процент
- Сколько всего правильных ответов дал за ВСЁ время в этом уровне
- Сколько всего попыток было за ВСЁ время
- Статус "пройден" (достигнут ли 70%)
- Получена ли уже золотая звезда (за 90%)
- Получен ли уже бриллиант (за 100%)
- Когда впервые начал уровень
- Когда последний раз играл

##### **B. По каждому слову в каждом уровне отдельно (обязательно сохраняем НАВСЕГДА):**
- Факт: "Это слово хоть раз было пройдено правильно в этом уровне"
- Сколько раз правильно отвечал на это слово
- Когда впервые правильно ответил на это слово
- Когда последний раз правильно ответил

##### **C. Вознаграждения (самое важное! Обязательно сохраняем НАВСЕГДА):**
- Отдельная таблица с уникальными ключами
- Каждая запись = факт получения денег
- Защита от дублирования

##### **НЕ сохраняем (временные данные):**
- Текущую позицию в уровне (какое слово показывать следующим)
- Порядок слов в текущей сессии
- Временные счётчики за текущий заход
